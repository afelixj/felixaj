Centering Elements Using inline-block

September 07, 2024 CSS by admin

Before the introduction of modern layout systems like Flexbox and CSS Grid, developers had to rely on more creative methods to center elements on a page. One of the common techniques used was the inline-block approach, which still comes in handy for various use cases today.

When to Use inline-block

The inline-block property is still quite useful, particularly for:

  • Buttons that need to be aligned centrally
  • Tag clouds or groupings of text elements
  • Situations where block elements need to behave like inline elements for layout purposes

Normally, block elements cannot be centered using the text-align: center property on their parent. However, by combining inline-block with other CSS tricks, you can center elements both horizontally and vertically.

The Technique

This technique involves applying inline-block to the inner element and using the :before pseudo-element on the parent container. Here's a basic example:

How It Works

  • Parent Container (.parent): Acts as the outer box with a set width and height. The :before pseudo-element ensures there is an invisible inline-block element to match the height of the parent.
  • Child Element (.child): The inner element you want to center. By giving it display: inline-block and using vertical-align: middle, you align it within the parent.

This setup achieves the vertical centering effect, which was tricky before Flexbox and Grid were available.
